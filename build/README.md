# App Icon

To customize the application icon, place your icon files here:

- **Linux**: `icon.png` (512x512 or 1024x1024)
- **macOS**: `icon.icns` 
- **Windows**: `icon.ico`

You can use tools like:
- [electron-icon-builder](https://www.npmjs.com/package/electron-icon-builder)
- Online converters to create icon files from a PNG

Default: The app will use Electron's default icon if custom icons are not provided.
